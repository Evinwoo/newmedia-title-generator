# newmedia-generator
社交媒体文案转换系统——人群/平台/场景/海内外

## 这是什么

你有一篇任何平台已经完稿的图文。
你需要的是：同一内容在红书、抖音、视频号、知乎、Twitter 上都能流通，
同时对不同人群（下沉市场 / 精致白领 / 宝妈）说不同的话，
还要能按需求输出海外本土语言版本。

这个项目就是解决这件事的工具和方法论，
关键点在于普通内容无法达到高效商业价值，搭配验证过的商业语言模型即可

## 三个核心模块

### 模块一：平台适配（Platform Adapter）
不同平台对内容的要求完全不同：
- 小红书：情绪共鸣 + 具体方法论 + 需求深挖 + 精致图文 + 关键词标签
- 抖音：前3秒钩子 + 口播脚本 + 字幕节奏 + BGM + 爆款热梗实时挖掘
- 视频号：熟人传播逻辑 + 中老年友好 + 私域运营传播适配性
- 知乎：原理拆解 + 数据支撑 + 专业感 + 长图文可拆解
- Twitter/X：海外本土语境 + 话题标签 + 简洁有力 

### 模块二：人群适配（Audience Adapter）
同一产品，不同人群关注点完全不同：
- 下沉市场：价格、简单、实用、口碑
- 精致白领：原理、品质、效率、身份认同
- 宝妈群体：使用场景、安全、省心、孩子相关
- Z世代：梗、反差、真实感、社交货币
- .....

### 模块三：多语言本地化（Localization）
不只是翻译，是文化语境的重新包装：
- 中文 → 英文（欧美平台：Instagram / Reddit / Medium）
- 中文 → 日文（日本平台）
- 中文 → 马来文（东南亚市场）
- .....

## Skills 体系（核心竞争力）

Skills 是本项目的灵魂。
每一条 Skill 都是从真实平台运营经验中提炼的写作规则，
AI 必须严格遵守这些规则才能输出符合平台调性的内容。

Skills 存放在 /skills 文件夹，按平台和人群分类管理。

## 产品形态规划

- 阶段一：GitHub 文档库（现在）- 沉淀方法论和 Skills
- 阶段二：网页工具 - 输入内容，选择目标平台/人群，一键输出
- 阶段三：小程序 / APP
